# MLRecorder para NVDA

MLRecorder agrega atajos globales de NVDA para grabar:

- audio del proceso enfocado,
- solo micrófono,
- mezcla del proceso enfocado + micrófono.

## Comandos por defecto

- NVDA+Shift+R: alternar grabación del proceso enfocado (iniciar/detener).
- NVDA+Shift+M: alternar grabación de micrófono (iniciar/detener).
- NVDA+Shift+X: alternar grabación mixta proceso + micrófono (iniciar/detener).
- NVDA+Shift+S: detener todas las grabaciones activas.
- NVDA+Shift+I: informar el estado actual.

Las grabaciones se guardan en:

- `%userprofile%\Documents\NVDA_MLRecorder`
